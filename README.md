# douyin_uid_converter

输入抖音号或者抖音昵称，返回抖音uid

Example:
```
 $ python3 main.py                                                                                                                                        [2020-01-15 11:08:11]
请输入抖音号:619462428
用户[619462428]的UID为:95901570542
```
